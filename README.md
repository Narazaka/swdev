# SWDEV

Runtime transfome by Service Worker PoC.

_DO NOT USE PRODUCTION_

## Install

```
$ deno install -qAf --unstable https://deno.land/x/swdev@0.0.1-alpha.7/swdev.ts

$ swdev init swdev-app
$ cd swdev-app
$ swdev serve
```

## Develop

$ swdev init myapp
$ cd myapp
$ swdev serve
